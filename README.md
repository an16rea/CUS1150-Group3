# CUS1150-Group3

This project involves creating a to-do list in Linux that represents tasks a person needs to complete over the course of one month. The purpose of the list is to practice creating and managing files in the Linux environment while organizing realistic daily responsibilities. The to-do list includes school work, personal tasks, appointments, chores, and other activities that could normally occur throughout a month. The List is written so another user could follow it day by day and complete the tasks in order. This project demonstrates the ability to create, edit, and manage text files in Linux while also showing how Linux can be used for everyday organization

# Tasks:

- Function for time stamps
- Function for priority level of tasks
- Function to display date task was added
- Function to display when tasks completed by
- Function to save data to a file
- Function to read data from a file
- Function to display  name of tasks
- Function to list tasks
- Function to create,remove, update tasks
- Function to mark when tasks are completed
- Function to create different list, Ex: school, work, etc..
- Function to return the next task to do across all to-do lists based on the closest deadline
- Function to auto mark a task on a list as high prioriy on the day before if its not done before

# Implementation Details
- Run through the Linux terminal (command-line interface)
- Language : C
- Environment : Ubuntu Virtual Machine
- Compile/run directly from terminal
